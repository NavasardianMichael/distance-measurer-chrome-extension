---

Distance Measurer gives you pixel-perfect spacing between any two elements on a webpage. No DevTools, no rulers, no hassle—just double-tap D and click.

Who is this for?
• Designers checking mockups against live sites
• Developers verifying layouts and alignment
• QA engineers hunting down spacing bugs
• Anyone who needs to answer "how many pixels between these two things?" without leaving the page

How it works:
• Double-tap D to turn on measurement mode
• Hover to preview elements with a dashed outline
• Click two elements to get instant measurements—dashed frames, connecting lines, and pixel values
• Tap the info button for a full breakdown: vertical and horizontal distances, top-to-top, bottom-to-bottom, and all the variants you need

Features:
• Pixel-perfect measurements in vertical and horizontal directions
• Works on any webpage, no setup required
• Keyboard-driven—double-tap D to activate, no toolbar or menus
• Customize colors on screen to match your workflow
• Non-intrusive—click outside when you're done and everything disappears
• Separate modal for more details about the distance between selected elements.
